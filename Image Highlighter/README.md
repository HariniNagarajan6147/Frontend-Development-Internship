📸 Live Image Gallery
📌 Project Overview:

The Live Image Gallery is a responsive web application developed using HTML, CSS, and JavaScript. It provides an interactive platform for displaying and browsing images in an organized and visually appealing layout. The project focuses on enhancing user experience through dynamic interactions, responsive design, and efficient DOM manipulation.

✨ Features:
🖼️ Interactive image gallery layout
📱 Fully responsive design for different screen sizes
⚡ Dynamic image display using JavaScript
🎨 Modern and user-friendly interface
🔍 Smooth navigation and image browsing experience
💻 Optimized layout for better usability

🛠️ Technologies Used:
HTML5 – Structure and content of the application
CSS3 – Styling, layouts, animations, and responsiveness
JavaScript (ES6) – DOM manipulation, event handling, and interactive functionality
