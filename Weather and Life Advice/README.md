🌤️ Weather & Life Advice Web Application:
📌 Project Overview:
The Weather & Life Advice App is an interactive web application developed using HTML, CSS, and JavaScript. The application provides users with weather-related information and personalized life advice based on different weather conditions, helping them plan their day more effectively.
This project enhanced my understanding of JavaScript logic, API integration concepts, DOM manipulation, event handling, and responsive web design.

✨ Features:
🌤️ Displays weather information based on user input
💡 Provides life advice and recommendations according to weather conditions
🔍 Search weather details by location
⚡ Dynamic content updates using JavaScript
📱 Responsive design for desktop and mobile devices
🎨 Clean and user-friendly interface
🌡️ Weather-based suggestions for daily activities

🛠️ Technologies Used:
HTML5 – Application structure and content
CSS3 – Styling, layout, and responsive design
JavaScript (ES6) – Weather logic, DOM manipulation, event handling, and dynamic updates
