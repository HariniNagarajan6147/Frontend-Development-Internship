Age Prediction & Calculator Web App
📌 Project Overview
This project is a combined Age Prediction and Calculator Web Application developed using HTML, CSS, and JavaScript as part of my Front-End Development Internship.

The application allows users to:
Enter their Date of Birth (Day, Month, and Year) to calculate their current age.
Perform basic arithmetic calculations using an integrated calculator.
Experience a clean and responsive user interface designed for better usability.

✨ Features
🎂 Age Prediction Module
User-friendly date input fields.
Calculates age based on the entered date of birth.
Displays the user's current age instantly.
Input validation for accurate results.

🧮 Calculator Module
Performs basic arithmetic operations.
Supports addition, subtraction, multiplication, and division.
Interactive and easy-to-use interface.
Real-time calculation display.

🛠️ Technologies Used
HTML5 – Structure and content of the web application.
CSS3 – Styling, layout, and responsive design.
JavaScript (ES6) – Application logic, age calculation, validation, and calculator functionality.
